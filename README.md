# Run on development

1. Clone repository
2. Run

```
yarn install
```

3. Install Nest CLI

```
npm i -g @nestjs/cli
```

4. Set up DB with docker

```
docker-compose up -d
```

## Used Stack

- MongoDB
- Nest
